This video shows how a person is able to incorporate media files along with the original document (pdf in this case) into a personalized mutable document. This mutable document is dynamic in nature and will change if any of the files attached to it are modified. However, the user can generate a static version of it (epub in this video).

This mutable document can be further modified by the user or it can be shared with the user's peers or even with the content creator of the original document.
